# Overview
This game runs on Python using the PYGAME pip. I've held this project on my storage for a long time, and finally decided it should be put into my github repository. This way anyone can stumble across the code and examine it themselves. This project was my final send off before graduating university. I spent many hours building, breaking, and learning with this project. I do apologize for any messy code, Uni student me kinda just slammed 99% of the code in a single py file without a care in the world.

## Project Title: Tank.IO - A 2D Top-Down Tank Battle Game

Description: A 2D top-down tank battle game where players navigate a maze-like arena, strategically maneuvering their tanks to outwit and outshoot their opponents. Players can choose between keyboard controls or gamepad input, adding versatility to the gameplay experience.
